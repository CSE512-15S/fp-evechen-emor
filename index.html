Running the Visualization

View our visualization here. For optimal viewing, use a Chrome browser and a screen with resolution greater than or equal 1280x800 so that the visualization fits onto a single screen without scrolling.

Our visualization requires: HTML file containing JS, HTML and CSS; D3.js; and student data files. These file are all included in the folder in our repository.
